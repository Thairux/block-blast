# Support

## What to Include
When reporting a problem, include:
- platform and device
- app version and flavor
- board size
- steps to reproduce
- whether cloud sync was configured
- diagnostics report or session snapshot if relevant

## Useful In-App Actions
- `Copy Session Snapshot`
- `Copy Diagnostics`

## Support Scope
Current support is focused on:
- gameplay defects
- persistence and restore issues
- sync problems
- platform-specific launch or audio failures
